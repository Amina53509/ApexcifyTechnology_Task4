# 📊 Internship Task 4: Basic Descriptive Statistics + Visualizations

## Overview
This project demonstrates how to perform **basic descriptive statistics** and create **visualizations** using Python.  
The dataset used is the **Iris dataset** (from Seaborn), but the workflow can be applied to other datasets such as `tips` or `titanic`.

## 🎯 Objectives
- Load a dataset using Seaborn.
- Calculate descriptive statistics:
  - Mean
  - Median
  - Minimum
  - Maximum
  - Standard Deviation
- Generate summary statistics with `describe()`.
- Identify missing values using `isnull().sum()`.
- Visualize dataset features with:
  - Histograms
  - Boxplots
  - Pairplots
    
## 🛠 Tools & Libraries
- **Python 3.x**
- **Pandas** → Data handling
- **Seaborn** → Dataset loading + visualization
- **Matplotlib** → Visualization
